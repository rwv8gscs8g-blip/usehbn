# HBN Roadmap

## Current stage

HBN is in the protocol-foundation stage.

## Near-term priorities

- stabilize protocol terminology
- improve examples and onboarding materials
- strengthen validation and traceability rules
- expand documentation for agent behavior and human review
- keep the runtime small and inspectable

## Future direction

HBN may evolve into a stronger platform and SaaS foundation, but that should happen only after the protocol, governance model, and documentation system are mature enough to support it without hidden behavior or unclear authority boundaries.
