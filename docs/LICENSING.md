# Licensing

## License choice

HBN is released under the GNU Affero General Public License v3.0.

## Why AGPL

HBN is an open protocol project intended to remain inspectable even when adapted for networked usage. AGPL is appropriate here because it discourages a pattern where protocol improvements remain privately deployed behind hosted systems without source disclosure.

## What the license supports

AGPL allows:

- use
- study
- modification
- redistribution

under the conditions of the license.

## Practical implication for hosted derivatives

If someone modifies HBN and runs that modified version for users over a network, AGPL requires them to provide access to the corresponding source code of that modified version.

## No extra claims

The license does not make HBN secure, correct, or complete. It governs openness and reciprocity, not engineering quality by itself.
