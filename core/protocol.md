# HBN Protocol

## Definition

HBN — Human Brain Net is an open protocol for safe, structured, and evolvable AI-assisted software engineering.

The protocol defines how a human-guided request should move from activation to structured execution while preserving traceability and human authority.

## Protocol Objectives

- make intent explicit
- make constraints visible
- surface risks early
- preserve human control
- document the basis for action

## Current Protocol Surface

The current repository formalizes these protocol elements:

1. semantic activation through `usehbn` or `use hbn`
2. structured intent with objective, constraints, risks, and validation requirements
3. local consent capture for contribution to the language
4. truth barrier warnings
5. guardian warnings and local logging

## Protocol Boundaries

HBN does not currently define:

- distributed execution
- hidden background processing
- remote policy enforcement
- autonomous system control

These exclusions are part of the protocol’s integrity, not temporary omissions.
