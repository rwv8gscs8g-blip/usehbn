# Usage Examples

## Basic activation

```text
usehbn analyze this repository
```

## Constraint-aware request

```text
use hbn refactor this module without changing public behavior
```

## Risk-sensitive request

```text
usehbn review the authentication flow and verify rollback coverage
```

## Consent capture example

```bash
usehbn "usehbn analyze this repository" --consent yes --allow-operation local_storage
```
