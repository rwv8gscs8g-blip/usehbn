# Prompt Samples

These examples show how HBN can be invoked as a protocol rather than as a vague request style.

## Repository analysis

```text
usehbn analyze this codebase, identify constraints, and list required validations
```

## Safe modernization

```text
use hbn modernize this VBA workflow without changing the exported file structure
```

## Release review

```text
usehbn review this release plan, flag unsupported claims, and require explicit validation steps
```
