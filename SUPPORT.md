# Support

## Project status

HBN is in its first public release. Support is community-oriented and best-effort.

## Where to ask for help

- use GitHub Issues for bugs, documentation problems, and protocol questions
- use pull requests for concrete proposed changes
- use repository discussions when that channel is available for higher-level design conversation
